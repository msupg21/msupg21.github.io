What is the difference between a ‘for’, ‘while’ and ‘do while’ loop?

A while loop statement allows to repeatedly executes a target statement as long as a given condition is true.

For loops allow you to efficiently write a loop that needs to excute a specific number of times. Example: for(initialization; condition;increment/ decrement) | for(int x = 1;x <=5,x++).

A do while loop looks similar to a while loop, except that a do while loop is guaranteed to execute at least one time which has its advantages.
